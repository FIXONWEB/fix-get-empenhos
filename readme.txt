Consultar registros de empenhos via API em portaltransparencia.gov.br

== Descrição ==

Este plugin habilita uma URL personalizada em seu site onde é possivel obter informaões sobre os registros de empenhos disponibilizados no portal da transparẽncia do governo federal.

Requisito:
1) Para ter acesso aos dados do portal da transparencia, se faz necessário que vocẽ crie uma chave de acesso antes de poder consultar. isso pode ser feito acessando xzxxzxx.

2) Depois de instalar este plugin, vocẽ poderá informar os parametros para que o mesmo possa funcionar corretamente. Acesse a pagina de configuração deste plugin e informe a chave de acesso do portal e tambem qual o favorecido você deseja fazer a consulta de dados de empenho.
